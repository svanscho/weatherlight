Weatherlight.

Led strip indicating the weather of the next day with colors.

It has been written with Arduino and ESP8266 in mind, but it should be quite portable so you can use this code in any other C++ project.

Getting started

You'll need:

  • ESP8266 (NodeMCU) or Arduino Uno (with Ethernet shield)
  • Openweathermap account: Openweathermap website
  • Google App Engine (GAE) account: GAE website

Make sure to check out the README to update the service configuration for your specific needs and any other additional details.
